# ChecksumFractal
## Prerequisites
- Java 21: https://adoptium.net/en-GB/temurin/releases/?version=21
## Compile
```powershell
PS ...\ChecksumFractal> javac -d target src/main/java/org/example/*.java
```
## Run
```powershell
PS ...\ChecksumFractal> java -cp target org.example.Main
```
